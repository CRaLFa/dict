# dict

English-Japanese dictionary for Bash

### Installation

```
$ git clone https://github.com/CRaLFa/dict.git
$ cd dict
$ make
$ sudo make install
```

### Usage

```
$ dict -h
Usage: dict [OPTION]... SEARCH_WORD

English-Japanese dictionary for Bash

Options:
    -h, --help          display this help and exit
    -w, --whole         match only whole SEARCH_WORD
```
